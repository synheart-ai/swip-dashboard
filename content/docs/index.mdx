# Welcome to SWIP

The **SWIP Dashboard** (Smart Wellness Intelligence Protocol) is the public transparency layer for Synheart's Wellness Impact Protocol. This documentation will help you understand how to integrate with the SWIP platform, work with our APIs, and interpret wellness metrics.

## What is SWIP?

SWIP is a wellness intelligence platform that:

- **Tracks Biometric Data**: HRV, heart rate, and stress metrics from wearables
- **Calculates Wellness Scores**: Real-time SWIP scores (0-100)
- **Provides Developer Tools**: API access for wellness app integration
- **Offers Transparent Rankings**: Global leaderboard for wellness apps

## Platform Overview

| Feature | Description |
|---------|-------------|
| **Transparency** | Publish anonymized SWIP scores, rankings, and population insights |
| **Developer Tools** | Register or claim apps, generate analytics API keys, and export reports |
| **Controlled Ingestion** | Only verified partners can stream biosignals into the platform |
